## 使用方法
将本项目clone至HoshinoBot\hoshino\modules下，在__bot__.py中加入auto_strategy  
！！注意！！本插件与自带查询指令冲突，在启用前请先修改HoshinoPath\hoshino\modules\priconne\query内的query.py和__init__.py删除有关指令及其帮助   

#### 首次使用需要先发送【更新攻略缓存】
